# 로또 기능 구현 목록

> Lotto 클래스 (로또)

    1. 유효한 로또인지 판별하는 기능

> LottoMachine 클래스 (로또기계)

    1. 금액과 알맞은 갯수의 로또를 생성하는 기능
    2. 로또 번호를 오름차순으로 정렬하는 기능
    3. 유효한 금액인지 판별하는 기능
    4. 유효한 당첨 번호인지 판별하는 기능
    5. 유효한 보너스 번호인지 판별하는 기능
    6. 로또와 당첨번호에서 일치하는 번호의 수를 반환하는 기능
    7. 로또들에서 일치하는 수로 등수를 정하는 기능
    8. 수익률을 계산하는 기능
    9. 보너스 번호를 확인하는 기능
